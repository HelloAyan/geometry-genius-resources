The full meaning of DOM is Document Object Model. DOM is and Application Programming Interface(API) that represents the HTML document. When a web page is loaded, the browser creates a Document Object Model of the page.
Most important objective for the Document Object Model is to provide a standard programming interface that can be used in a wide variety of environments and applications.

There are 5 ways in that we can select elements in a DOM. Such as -
1. getElementsByTagName(). By using tag name we can use this selector
2. getElementsByClassName(). By using class name we can use this selector
3. getElementById(). By using Id we can use this selector
4. querySelector(). We can change one element using this selector
5. querySelectorAll(). We can change multiple element using this selector

Event bubbling is a method of event propagation in the HTML DOM API when an event is in an element inside another element, and both elements have registered a handle to that event.

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.
A callback's purpose is to execute code in response to an event. These events can be like mouse clicks; with callback, we can add text at the end of each statement like"execute this code every time the user clicks a key on the keyboard."
